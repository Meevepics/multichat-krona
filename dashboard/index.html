🔑 TIKTOK_SESSION_ID
  1. Abre tiktok.com en Chrome e inicia sesión
  2. Pulsa F12 → pestaña Application
  3. Panel: Cookies → https://www.tiktok.com
  4. Busca la cookie sessionid → copia el valor
  5. En Render/Railway:
    TIKTOK_SESSION_ID = (el valor)
  6. Reinicia el servidor

⚠️ El sessionid caduca cada ~30 días.
